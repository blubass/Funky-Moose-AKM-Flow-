FUNKY MOOSE RELEASE FORGE - KURZANLEITUNG FUER MAC

1. ZIP-Datei entpacken
   - Doppelklick auf "Funky Moose Release Forge.zip"

2. App an einen passenden Ort ziehen
   - z. B. in "Programme" oder auf den Schreibtisch

3. App starten
   - Doppelklick auf "Funky Moose Release Forge.app"

WICHTIG:
Es ist keine Python-Installation noetig.
Es muessen keine Pakete installiert werden.

WENN MACOS DIE APP BLOCKIERT:

Variante A:
1. Rechtsklick auf "Funky Moose Release Forge.app"
2. "Oeffnen" waehlen
3. Im naechsten Dialog erneut "Oeffnen" bestaetigen

Variante B:
1. App einmal normal starten
2. Dann zu "Systemeinstellungen > Datenschutz & Sicherheit"
3. Unten auf "Trotzdem oeffnen" klicken

NUR WENN NOETIG - TERMINAL-WEG:

xattr -dr com.apple.quarantine "/Pfad/zu/Funky Moose Release Forge.app"

Beispiel:
xattr -dr com.apple.quarantine "/Applications/Funky Moose Release Forge.app"

DATEN DER APP:
- ~/akm_assistant/data.json
- ~/akm_assistant/data_backup.json
- ~/akm_assistant/lang.txt
- ~/akm_assistant/settings.json

UPDATE:
1. Alte App schliessen
2. Neue ZIP entpacken
3. Alte App durch die neue ersetzen
4. Daten bleiben erhalten
