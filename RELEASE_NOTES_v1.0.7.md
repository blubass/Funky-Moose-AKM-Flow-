# v1.0.7 — Forge UI Polish & Release Infrastructure

Diese Version konzentriert sich auf die optische und funktionale Verfeinerung („Polish“) der Forge-Oberfläche sowie die Stabilisierung des Desktop-Build-Workflows.

## Highlights

- **Umfassender UI Polish**: Überarbeitung von Buttons, Dialogen und Tabs für ein konsistenteres „Premium“-Gefühl.
- **Optimierte Tab-Architektur**: Schnellere Ladezeiten und sanftere Übergänge durch verbessertes Lazy Loading.
- **Refresh Desktop Flow**: Der Build-Prozess für macOS wurde weiter automatisiert und stabilisiert.
- **Verbesserte Status-Visualisierung**: Der Systemstatus ist nun in allen Werkzeugen deutlicher erkennbar.

## Änderungen im Detail

### UI & UX
- Neue, hochwertigere Button-Styles im gesamten System.
- Überarbeitete Header-Sektion für bessere Orientierung.
- Konsistente Anwendung des industriellen Designs über alle Widgets hinweg.
- Verbesserte Drag & Drop Visualisierung.

### Core & Workflow
- Update des State-Managements für stabilere Hintergrundprozesse.
- Verfeinerung der i18n-Integration in allen UI-Komponenten.
- Bereinigung redundanter Delegate-Aufrufe in den Controllern.

## Build-Artefakte
- `Funky Moose Release Forge.app` (macOS)
- `Funky Moose Release Forge.zip` (Windows-kompatibel via CI)
